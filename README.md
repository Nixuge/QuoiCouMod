# Mod template
A simple, easy to setup 1.8.9 Forge mod template.

## Just run :

### Linux:
```sh
./setup.py
./gradlew setupDecompWorkspace
# then to run a debug client
./gradlew runClient
# or to build a mod jar
./gradlew build
```

### Windows:
```sh
python setup.py
gradlew.bat setupDecompWorkspace
# then to run a debug client
gradlew.bat runClient
# or to build a mod jar
gradlew.bat build
```
